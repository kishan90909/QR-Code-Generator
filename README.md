# 🌟 Overview

The **QR Code Generator** is a simple and interactive web application that allows users to instantly generate QR codes for any URL or text input.

Users can enter a website link, select the QR code size, and generate a downloadable QR code image within seconds. The application features a clean user interface, loading indicators, and a download option for saving the generated QR code.

This project demonstrates **frontend development skills including DOM manipulation, event handling, dynamic UI updates, and integration with QR code generation logic.**

---

# ✨ Key Features

### 🔳 QR Code Generation

* Instantly generate QR codes from a URL or text input.
* Select different QR code sizes before generation.

### ⬇️ Download QR Code

* Generated QR codes can be downloaded as an image.

### ⚡ Interactive UI

* Loading spinner while generating QR code.
* Generate button disables after generation.
* Re-generate option resets the interface.

### 📱 Responsive Layout

* Clean 3-column layout:

  * QR illustration
  * Input form
  * Generated QR code

# 🚀 Installation & Setup

### 📋 Prerequisites

You only need a web browser.

### 🛠️ Step 1: Clone the Repository

```bash
git clone https://github.com/yourusername/qr-code-generator.git
```

### 🛠️ Step 2: Navigate to the Project Folder

```bash
cd qr-code-generator
```

### 🛠️ Step 3: Open the Application

Simply open the `index.html` file in your browser.

```bash
index.html
```

---

# 💻 Usage

1. Enter a **website URL** in the input field.
2. Select the **QR code size**.
3. Click **Generate QR Code**.
4. The QR code will appear in the **result section**.
5. Click **Download QR Code** to save the image.
